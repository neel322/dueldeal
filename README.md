#ğŸ§  How It Works

The application uses LocalStorage as a lightweight database:

Products stored in localStorage

Cart items stored separately

Wishlist data stored per user

Orders saved after checkout

Logged-in user session stored locally

On page load, JavaScript dynamically:

Injects the global navbar

Loads product data

Updates cart counters

Displays user-specific content

The system initializes with sample products if no data exists.
